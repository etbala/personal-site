## Running Locally

#### Clone the Repository
```bash
git clone https://github.com/etbala/personal-site.git
cd personal-site
```

#### Install Angular

```bash
npm install -g @angular/cli
```

#### Local Development

```bash
ng serve
```

#### Building

```bash
ng build
```
